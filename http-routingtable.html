HTTP Routing Table

/device | /participant | /service | /test | /user
 
/device
GET /device/assets Get device assets
GET /device/devices Get connected device information
GET /device/login Device login
GET /device/logout Device logout
GET /device/participants Get device associated participants information
GET /device/register Register a new device in the server (token based)
GET /device/sessions Query device sessions
GET /device/sessions/events Get events for a specific session
POST /device/devices Update current device information
POST /device/register Register a new device in the server (certificate based)
POST /device/sessions Update / create a session
POST /device/sessions/events Create / update session events
POST /device/status Update current device status
DELETE /device/sessions Delete a session
DELETE /device/sessions/events Delete session events
 
/participant
GET /participant/assets Get participant assets
GET /participant/devices Get associated participant devices
GET /participant/login Login a participant with username / password
GET /participant/logout Participant logout
GET /participant/participants Get current participant informations
GET /participant/refresh_token Refresh participant dynamic token
GET /participant/sessions Get participant sessions
POST /participant/participants Update current participant informations
POST /participant/sessions Create / update a session
